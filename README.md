Native C++ AI music generation powered by the [acestep.cpp](https://github.com/audiohacking/acestep.cpp) GGUF-native inference engine + [acestep-ui](https://github.com/audiohacking/acestep-cpp-ui)

## Features

- No Python dependency — standalone C++ + Node.js
- GGUF quantization support (Q4–Q8, BF16)
- Multi-GPU: CUDA, Vulkan (Linux), Metal/MPS (macOS Apple Silicon)
- Generate music from text prompts or structured lyrics
